Floating action button

One persistent, screen-level create action — Ny lönekörning, Ny anställd. It floats above the content, bottom-right, clear of the tab bar. Circular when the icon is self-evident; extended with a label when the action needs naming or on a first-run screen. Accent-mint CTA fill with the brand glow; a Signature variant exists for secondary contexts.

Riktlinjer

Dos & don'ts

Do
  • Reserve the centre + for the one create action (Absence / Expense).
  • Flip it from mint (--status-badge) to --green + × while the hub is open.
  • Keep it 56px and lifted above the tab bar.
Don't
  • Give the + a destination/selected state — it is not a tab.
  • Add a label under the +.
  • Use it for more than the single create action.
